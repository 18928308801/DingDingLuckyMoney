# DingDingLuckyMoney

钉钉抢红包插件，基于 Xposed 框架，速度快得令人发指。为避免过度拉仇恨，请谨慎使用。

添加了防撤回功能。

Merge code from [DingDingUnrecalled](https://github.com/veryyoung/DingDingUnrecalled)
